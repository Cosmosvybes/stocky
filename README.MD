# Stocky

Stocky  is a simple Command Line Interface inventory management system that allows users to add, remove, update, and log sales for products in their stores.

## Features

- Add new products to the store
- Remove products from the store
- Update product prices
- Log sales for products
- Get list of products in a store
- Get list of stores


## Commands 

 - add
 - products
 - del
 - price
 - stores
 - report

## Usage 
- Ensure you have Dotnet version 10.0.0 installed on on your machine
- run with the command `` dotnet run ``

## Example
```
Prompt Stocky > add
```
